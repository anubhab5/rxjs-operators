Defn
Creates an Observable from an Array, an array-like object, a Promise, an iterable object, or an Observable-like object.

Theory

  • from converts various other objects and data types into Observables.
  • It also converts a Promise, an array-like, or an iterable object into an Observable that emits the items in that promise, array, or iterable.
  • A String, in this context, is treated as an array of characters.
  • Observable-like objects (contains a function named with the ES2015 Symbol for Observable) can also be converted through this operator.

How It Works

Creates Observables from Array

return from([1, 2, 3]);

Output: when subscribed will get 1, 2 , 3
Creates Observables from Promise
const promise = new Promise((resolve, reject) => return from(promise);
Creates Observables from Iterables
function* generateItems() {{ "{" }} yield 'Item 1'; yield 'Item 2'; {{ "}" }}
return from(generateItems());
Creates Observables from Observables
// Emit a value every second, for 5 seconds
const sourceObservable = interval(1000).pipe(take(5));
// Convert the Observable to another Observable using from return
from(sourceObservable);
